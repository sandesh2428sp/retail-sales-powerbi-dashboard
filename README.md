Retail Sales Performance Dashboard | Power BI

Project Overview
This project presents an interactive Retail Sales Performance Dashboard built using Power BI. The objective of this dashboard is to transform raw retail sales data into meaningful business insights for data-driven decision-making.

The dashboard highlights key performance indicators (KPIs) and visual trends to analyze revenue, profitability, customer behavior, and regional performance.

---

Key Features

- 📈 Monthly Sales Trend Analysis  
- 📊 Category-wise Sales Performance  
- 🌍 City-wise Revenue Distribution  
- 💳 Payment Method Analysis  
- 📌 KPI Cards for:
  - Total Revenue
  - Total Quantity Sold
  - Gross Income
  - Average Rating

- 🔎 Interactive Slicers for dynamic filtering

---
Tools & Technologies Used

- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Excel Dataset  

---

 Dataset

Supermarket Sales Dataset containing transactional records including:
- Date
- Product Line
- City
- Payment Method
- Quantity
- Total Revenue
- Gross Income
- Customer Rating
